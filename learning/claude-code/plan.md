# Claude Code 学习计划

## Mission

理解 Claude Code 的核心架构和工具体系，能够自主设计、调试和优化自己的 skill 工作流，让它在小说创作中更稳定、更高效。

---

## 学习者画像

- **经验**: 1 个月，使用 Claude Code 写小说
- **已知**: 会安装 MCP、skill、plugin，用过 5+ AI Agent
- **目标**: 从"会用"进化到"精通"，理解底层机制
- **偏好**: 先原理，后例子
- **时间**: 每天可学

---

## 学习路径（80/20 原则）

核心 20% 决定 80% 实践价值的内容：

### 第一阶段：理解底层（Session 1-3）
1. **核心循环** — Claude Code 是怎么工作的（提示 → 分析 → 决策 → 行动）
2. **上下文管理** — 为什么有时会"忘记"，如何控制
3. **工具体系** — 8 个核心工具各做什么

### 第二阶段：掌握配置（Session 4-6）
4. **记忆系统** — CLAUDE.md 的层级和作用
5. **Skill 机制** — 你已经在用的东西，理解它为什么有效
6. **Hook 系统** — 事件自动化，让工作流更稳定

### 第三阶段：进阶优化（Session 7-9）
7. **Agent 架构** — 多智能体协作
8. **调试技巧** — 遇到问题时如何自己解决
9. **工作流设计** — 为小说创作设计最优流程

## 学习原则

1. **学习者做思考** — 我提问，你回答，不是我念文档
2. **具体先于抽象** — 先看例子，再总结原理
3. **间隔复习** — 每次开始先复习上次内容
4. **费曼检验** — 用简单的话解释概念
5. **渐进难度** — 答对了就加难度，答错了就降难度
